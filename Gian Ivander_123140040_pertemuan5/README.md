# Sistem Manajemen Perpustakaan Sederhana

Program ini merupakan implementasi sederhana dari sistem manajemen perpustakaan menggunakan konsep OOP Python: **abstract class**, **inheritance**, **encapsulation**, dan **polymorphism**. Program dibuat untuk memenuhi tugas praktikum OOP.

## Fitur Utama
- Abstract class `LibraryItem` sebagai dasar semua item.
- Subclass `Book` dan `Magazine` mengimplementasikan method abstract.
- Class `Library` untuk mengelola koleksi item perpustakaan.
- Encapsulation dengan atribut protected & private.
- Penggunaan `@property` pada beberapa atribut.
- Program dapat:
  - Menambahkan item ke perpustakaan
  - Menampilkan seluruh item
  - Mencari item berdasarkan judul
  - Mencari item berdasarkan ID
  - Menghapus item berdasarkan ID

## Cara Menjalankan Program 
1. Pastikan Python 3.8+ sudah terinstall.
2. Buka terminal di folder proyek.
3. Jalankan:

```bash
python library_system.py
```bash
